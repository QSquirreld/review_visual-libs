# Демонстрация применения библиотек для визуализации

Применение библиотек matplotlib, Seaborn, Plotly для демонстрации визуализации данных из датафреймов Pandas.

## Описание

__Код содержит визуализацию данных из трёх датасетов:__
* California Housing Prices (housing.csv), который содержит данные о медианных ценах на жилье в районах Калифорнии по данным переписи населения 1990 года;
* Jena Climate Dataset (jena_climate_2009_2016.csv), который содержит метеорологические данные в период несскольких лет;
* Titanic (классика), который содержит информацию о пассажирах Титаника.

__Демонстрируются различные виды графиков, их параметры и методы библиотеки Seaborn, а также более интерактивная альтернатива - Plotly.__

__Кроме того, был выполнен краткий анализ на основе графиков.__

## Требования

- Код был написан на Python 3.12
- Зависимости из файла `requirements.txt`
