# review_visual-libs

Небольшой учебный проект с обзором популярных библиотек визуализации данных в Python.  
Цель репозитория — сравнить основные инструменты для построения графиков, посмотреть, в чём их сильные стороны и в каких задачах каждый из них удобнее использовать.

В проекте рассматриваются библиотеки, которые часто применяются в анализе данных и Data Science: для базовых графиков, разведочного анализа, наглядного сравнения признаков и более удобного представления результатов.

Этот репозиторий можно воспринимать как вспомогательный обзорный проект: не про отдельную ML-задачу, а про инструменты, которые полезны при EDA, аналитике и подготовке понятных визуализаций для дальнейшей работы с данными.

## Связанные проекты

Этот проект посвящён обзору библиотек визуализации и связан с другими учебными репозиториями по анализу данных.  
Его можно рассматривать как вспомогательный проект: он не столько про отдельную задачу, сколько про инструменты, которые полезны в EDA и моделировании.

### [regression_tasks](https://github.com/QSquirreld/regression_tasks)
Проект по регрессионным задачам, где визуализация может использоваться для анализа признаков, проверки зависимостей и интерпретации результатов модели.  
Этот репозиторий показывает, где подобные инструменты применяются уже на практике.

### [my-first-data-cleaning](https://github.com/QSquirreld/my-first-data-cleaning)
Проект про очистку данных.  
После подготовки таблицы следующим естественным шагом часто становится визуальный анализ, поэтому эти проекты хорошо связаны между собой.

### [my-first-EDA](https://github.com/QSquirreld/my-first-EDA)
Самая близкая по смыслу связка: визуализация — одна из ключевых частей EDA.  
Если нужен пример того, где графики используются не сами по себе, а как инструмент анализа данных, стоит посмотреть этот репозиторий.

### [review_SQL](https://github.com/QSquirreld/review_SQL)
Проект по SQL, который дополняет работу с данными на более раннем этапе.  
SQL помогает получить и подготовить данные, а визуализация — уже исследовать и объяснить их содержательно.
